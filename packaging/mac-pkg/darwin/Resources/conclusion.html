Ritchie CLI

Thank you for installing Ritchie CLI.

Run the CLI

Open a new terminal and run the following command to get started with Ritchie:

  $ rit init

Uninstall Ritchie CLI

Run the following command to uninstall Ritchie.

  $ sudo bash /Library/rit/uninstall.sh
